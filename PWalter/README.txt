
Projeto Walter Thummel - Versão Web com Streamlit

Requisitos para rodar localmente:
- Python 3.x instalado
- Streamlit instalado (pip install streamlit)

Como executar localmente:
1. Extraia o arquivo ZIP.
2. Abra o terminal na pasta extraída.
3. Rode: streamlit run app.py
4. O app abrirá no navegador automaticamente.

Como publicar no Streamlit Cloud:
1. Crie uma conta no https://streamlit.io/cloud
2. Faça upload do projeto (app.py e requirements.txt) no Streamlit Cloud.
3. O Streamlit Cloud gera um link público para acessar o app.

Divirta-se!
